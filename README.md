# RSSI 多边室内定位系统

基于 WiFi-RSSI 信号强度实现的**室内二维多边定位系统**，采用**发射节点 + 接收节点 + 主控节点**三级分布式硬件架构，依托 BLE 蓝牙透传完成跨设备数据交互，结合对数路径损耗模型与最小二乘算法解算坐标，是一套低成本、易部署的室内定位解决方案。

## 一、项目结构

本项目包含三套独立固件工程，分别对应不同功能硬件，按需烧录至对应设备即可运行：

- `RSSI_emitter/`：发射节点程序（烧录至 ESP32-C3）
- `RSSI_receiver/`：接收节点程序（在宏定义修改从机名，分别烧录至三台 ESP32-C3）
- `RSSI_compute/`：主控解算节点程序（烧录至 ESP32-P4）

> 补充说明：蓝牙透传底层协议、数据帧格式、通信逻辑详见独立仓库。

## 二、整体工作流程

1. **发射节点**：ESP32-C3 以 WiFi AP 模式持续广播无线信号，作为定位目标；
2. **接收节点**：三台 ESP32-C3 同步扫描发射节点的 WiFi 信号，实时采集 RSSI 信号强度；
3. **距离换算**：接收节点通过对数路径损耗模型，将 RSSI 值转换为与发射节点的实际物理距离；
4. **数据上传**：接收节点通过 BLE 蓝牙透传，将距离数据实时上报至 ESP32-P4 主控；
5. **坐标解算**：主控节点汇总三组距离数据，基于最小二乘法求解超定方程组，计算出发射节点二维坐标；
6. **结果输出**：主控通过串口打印定位坐标、原始距离等信息，完成实时室内定位。

## 三、定位算法原理

### 3.1 RSSI 转距离：对数路径损耗模型

室内无线信号随传播距离增加呈对数规律衰减，利用该特性将采集到的 RSSI 强度换算为物理距离，公式如下：

$$
d = d_0 \times 10^\frac{RSSI_0 - RSSI}{10 \times n}
$$

#### 参数释义

| 参数 | 含义 | 常规取值 |
| ---- | ---- | ---- |
| $d$ | 接收节点与发射节点的实际距离（单位：m） | 算法输出值 |
| $d_0$ | 参考校准距离 | 固定取 1 m |
| $RSSI_0$ | 参考距离 $d_0$ 处实测的信号强度 | 现场校准得到（单位：dBm） |
| $RSSI$ | 接收节点实时采集的 WiFi 信号强度 | 硬件采样值（单位：dBm） |
| $n$ | 环境衰减因子 | 室内场景：2.0 ~ 4.0，根据现场环境微调 |

### 3.2 多边定位几何模型

系统以**三台接收节点作为定位锚点**，提前通过宏定义固定锚点二维坐标 $(x_i, y_i)$；设发射节点待求解坐标为 $(x, y)$，锚点到发射节点的测量距离为 $d_i$。

根据平面两点间距离公式，建立方程组：

$$
\begin{cases}
(x - x_1)^2 + (y - y_1)^2 = d_1^2 \\
(x - x_2)^2 + (y - y_2)^2 = d_2^2 \\
(x - x_3)^2 + (y - y_3)^2 = d_3^2
\end{cases}
$$

由于 RSSI 测量存在环境噪声、多径干扰，上述方程组为**超定方程组**，无严格解析解，因此采用**最小二乘法**求解全局最优坐标。

### 3.3 最小二乘法求解过程

1.  **线性化处理**
    将二次方程组两两作差，消去二次项，化简得到标准线性方程组：
    $$
    \boldsymbol{A}\boldsymbol{X} = \boldsymbol{B}
    $$
    其中：
    - $\boldsymbol{A}$：由三个锚点坐标构成的系数矩阵；
    - $\boldsymbol{X} = \begin{bmatrix} x \\ y \end{bmatrix}$：待求解的目标二维坐标；
    - $\boldsymbol{B}$：由距离、锚点坐标组合得到的常数向量。

2.  **最小二乘最优解**
    对线性方程组求解最小二乘解，公式如下：
    $$
    \boldsymbol{X} = (\boldsymbol{A}^\mathrm{T} \boldsymbol{A})^{-1} \boldsymbol{A}^\mathrm{T} \boldsymbol{B}
    $$
    - $\boldsymbol{A}^\mathrm{T}$：矩阵 $\boldsymbol{A}$ 的转置矩阵；
    - $(\boldsymbol{A}^\mathrm{T} \boldsymbol{A})^{-1}$：矩阵 $\boldsymbol{A}^\mathrm{T}\boldsymbol{A}$ 的逆矩阵。

通过上述公式计算，得到误差最小的最优坐标 $(x, y)$，完成定位解算。

## 四、系统特点

1. **硬件低成本**：基于通用 ESP32 系列芯片开发，无需 UWB、专业定位基站等高价设备；
2. **部署简单**：仅需固定三个接收锚点坐标，无需复杂布线，适配各类室内场景；
3. **抗干扰优化**：对原始 RSSI 数据做滑动滤波，抑制信号抖动，提升定位稳定性；
4. **实时性良好**：默认定位周期 2 s，满足室内目标连续追踪需求；
5. **架构解耦**：发射、接收、解算节点功能独立，依托蓝牙透传实现分布式数据交互，扩展性强。
